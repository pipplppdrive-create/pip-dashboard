# Fase 1 — Inisialisasi Project Baru

Baca kembali seluruh dokumen di folder `Docs`.

Kerjakan hanya Fase 1.

Tugas:

1. Pilih stack terbaik untuk aplikasi ini.
2. Inisialisasi project baru di folder utama.
3. Gunakan TypeScript strict.
4. Siapkan lint dan formatter.
5. Siapkan unit test dan E2E test.
6. Siapkan struktur route:
   - `/login`
   - `/dashboard`
   - `/pekerjaan`
   - `/admin`
7. Siapkan struktur feature, component, service, type, validation, dan test.
8. Buat `.env.example` tanpa secret.
9. Siapkan placeholder route sederhana.
10. Jangan membangun UI halaman lengkap.
11. Jangan membuat database penuh.
12. Jangan mengerjakan Dashboard, board, atau Admin.
13. Jangan menggunakan kode aplikasi lama.

Jalankan:

- lint;
- type-check;
- test dasar;
- build.

Laporkan:

- stack;
- alasan;
- dependency;
- struktur folder;
- route;
- file yang dibuat;
- hasil lint;
- hasil type-check;
- hasil test;
- hasil build;
- keputusan yang perlu disetujui.

Berhenti setelah Fase 1.
