# Fase 2 — Fondasi UI/UX

Baca seluruh dokumen fungsi.

Claude menentukan rekomendasi UI/UX terbaik untuk aplikasi pemerintah yang modern, profesional, mudah digunakan, responsif, dan nyaman ditampilkan pada TV.

Kerjakan:

- design system;
- color tokens;
- typography;
- spacing;
- app shell;
- navigation;
- header;
- button;
- input;
- select;
- modal/feedback;
- loading;
- empty;
- error;
- success state;
- responsive foundation;
- accessibility foundation.

Jangan mengerjakan Dashboard, board, atau Admin secara lengkap.

Uji minimal:

- 1920×1080;
- 1440×900;
- 768×1024;
- 390×844.

Laporkan keputusan UI/UX dan screenshot.

Berhenti setelah Fase 2.
